第4课 · 准则四：适量吃鱼、禽、蛋、瘦肉

《中国居民膳食指南（2022）》注册营养师考点串讲｜温故（准则一二三）→ 新知 → 口诀 → 易错 → 解题 → 提问 → 改编题 → 费曼 → 预告

一、温故（上三课要点快速过）

考点一句话核心关键数字
平衡膳食宝塔五层：谷薯→蔬果→畜禽鱼蛋奶→大豆坚果→油盐盐<5g、油25–30g、奶300–500g
准则一 食物多样谷类为主、每天12种/周25种谷类200–300g（全谷杂豆50–150g、薯类50–100g）
准则二 吃动平衡能量天平、健康体重BMI 18.5–23.9；腰围男≥90女≥85；6000步/周150分
准则三 蔬果奶类全谷大豆餐餐有蔬菜、天天有水果、常吃奶类全谷蔬菜≥300g（深色½）、水果200–350g、奶≥300ml、坚果≈10g

今日目标：在"食物多样"的大盘子里，把动物性食物这一格讲透——吃多少、吃什么、怎么吃。

二、导入（一个案例）

案例：高三男生小明"无肉不欢"——顿顿红烧肉、早餐培根三明治、周末炸鸡，几乎不吃鱼，还嫌蛋黄"胆固醇高"只吃蛋白。最近体检腰围 92cm、血脂偏高。
老师提问：小明的吃法，踩了准则四的哪些"坑"？学完今天这一课，你能给他开出几条整改单吗？

三、新知讲解：准则四 适量吃鱼、禽、蛋、瘦肉

1. 先记"量"——总量与每周配额

• 每天：鱼、禽、蛋类和瘦肉合计 120～200 g（这是所有动物性食物的总量，不是单指瘦肉）。

• 每周三类各自配额（注意是"各"300～500g，不是三者加起来）：

  • 鱼（水产品）：300～500 g，最好每周吃 2 次；
  • 蛋：300～350 g ≈ 每周 6～7 个 ≈ 每天 1 个；
  • 畜禽肉：300～500 g（其中畜肉建议 ≤500 g/周，≈每天 ≤70 g）。
图1｜每天 120–200g 是全部动物性食物的总量；每周鱼、蛋、畜禽肉各自 300–500g。

2. 六条核心推荐

官方原文 5 条

  1. 鱼、禽、蛋类和瘦肉摄入要适量，平均每天 120～200g。
  2. 每周最好吃鱼 2 次或 300～500g，蛋类 300～350g，畜禽肉 300～500g。
  3. 少吃深加工肉制品。
  4. 鸡蛋营养丰富，吃鸡蛋不弃蛋黄。
  5. 优先选择鱼，少吃肥肉、烟熏和腌制肉制品。

考试常用 6 条拆分

  1. 适量摄入，平均每天 120～200g；
  2. 每周鱼 2 次/300–500g、蛋 300–350g、畜禽肉 300–500g；
  3. 少吃深加工肉制品；
  4. 吃鸡蛋不弃蛋黄；
  5. 优先选择鱼和禽（白肉优先）；
  6. 吃畜肉宜选瘦肉，少吃肥肉、烟熏和腌制肉制品。
较真提示：官方第 5 条写"优先选择鱼"，教材常扩展为"优先鱼和禽"——因为我国居民鱼、禽肉摄入过低、畜肉过高，调整方向一致，考试两种说法都对，记住"白肉（鱼禽）优先"即可。

3. 底层逻辑：为什么这样推荐？（关键事实·证据）

  • 动物性食物是优质蛋白、脂肪、维生素 A、B₂、B₁₂、烟酸、铁、锌、硒的高贡献来源。
  • 现状问题：畜肉过高、鱼禽过低 → 指南才强调"优先鱼禽、控制畜肉"。
  • 健康证据：增加鱼类→降低全因死亡、脑卒中风险；适量禽肉、鸡蛋→与心血管疾病风险无明显关联；过量畜肉→增加 2型糖尿病、结直肠癌、肥胖风险；烟熏肉→增加 胃癌、食管癌风险。

4. 白肉 vs 红肉（鱼禽 vs 畜）——选肉"优先序"

图2｜白肉优先、红肉选瘦；鱼禽蛋，红肉瘦，加工熏腌靠边走。
图3｜鱼富含 ω-3（DHA/EPA），优先选择鱼，护心又护脑。

5. 鸡蛋与蛋黄——破除"胆固醇误区"

图4｜蛋黄富集卵磷脂、胆碱、维生素、矿物质与叶黄素；膳食胆固醇对血胆固醇影响小，每天 1 个蛋与心血管风险无关。

6. 加工 / 熏腌肉制品——明确"少吃"

  • 深加工肉制品（香肠、培根、火腿、腊肉、肉罐头）：高盐、油脂过度氧化、常含亚硝酸盐 → 长期食用有风险，应尽量少吃。
  • 拓展（IARC 分级，常考）：国际癌症研究机构将加工肉类列为 1 类致癌物（明确致癌）、红肉列为 2A 类（可能对人类致癌）——这正是"少吃熏腌和深加工肉"的科学依据。
图5｜新鲜鱼禽（绿）＞红肉适量（橙，2A类）＞加工肉少吃（红，1类）。香肠、培根、火腿、腊肉属加工肉。

其他：动物内脏建议每月 2～3 次、每次不过多，不必追求"山珍海味"。

图6｜把"畜肉过多、鱼禽不足、弃蛋黄"调成"优先鱼禽、不弃蛋黄、控总量"。

四、口诀（背下来就是分）

① 总量/周配额："一天一二两（120–200g），鱼两顿（2次/周）、蛋七个（300–350g）、畜禽各半斤（各300–500g）。"

② 六条："适（量）每（周配额）深（加工少吃）蛋（黄不弃）优（先鱼禽）瘦（肉）。"

③ 选肉优先序："鱼禽蛋，红肉瘦；加工熏腌靠边走。"

④ 蛋黄："蛋黄是个宝，弃之太可惜；卵磷胆碱好，眼脑都需要。"

五、易错点（6 个，考试最爱挖坑）

易错1：把"鱼 300–500g + 畜禽肉 300–500g"误当成合计 300–500g → 实为各类各自 300–500g/周。
易错2：把"平均每天 120–200g"当成单指瘦肉 → 它是鱼、禽、蛋、瘦肉全部动物性食物的总量。
易错3：弃蛋黄误区（怕胆固醇不吃蛋黄）→ 应不弃蛋黄。
易错4：把"优先选择鱼/禽"记反成"优先红肉" → 白肉（鱼禽）优先。
易错5：把加工肉当普通肉忽略致癌风险 → 深加工/烟熏腌制肉要少吃（IARC：加工肉 1 类）。
易错6：混淆"畜禽肉 300–500g/周"与"畜肉 ≤500g/周" → 畜禽肉（合计）300–500g/周，其中畜肉不宜超过 500g/周，鱼要≥2次。

六、三步解题法

第一步·读题定位：判断考点属于"量 / 种类优选 / 蛋黄 / 加工肉 / 白红肉"哪一类。

第二步·套数字原则：120–200g/天；鱼 300–500/周（2次）；蛋 300–350/周；畜禽 300–500/周；不弃蛋黄；优先鱼禽；少吃加工熏腌。

第三步·辨陷阱：看是否偷换"合计/各类"、是否把误区当正确（弃蛋黄、红肉优先）、是否忽略致癌分级。

七、课堂提问（师生互动）

问1：小明每天吃 2 个蛋黄，会"胆固醇爆表"吗？
答：不会。膳食胆固醇对血胆固醇影响小；不弃蛋黄是正解。但前提是动物性食物总量别超 120–200g/天，整体均衡。
问2：有人说"我每周吃 500g 牛肉 + 500g 鸡肉，刚好符合畜禽肉 300–500g"——对吗？
答：错！畜禽肉（合计）建议 300–500g/周，500+500 远超上限。
问3：香肠、培根算"适量吃瘦肉"吗？
答：不算。它们是深加工肉制品，准则四明确要求"少吃"。
问4：鱼和畜禽肉每周都是 300–500g，每天到底吃多少？
答：分散到各餐（最好每餐有肉、每天有蛋），每天动物性食物总量 120–200g。
