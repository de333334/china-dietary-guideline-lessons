第5课 · 准则五：少盐少油，控糖限酒

《中国居民膳食指南（2022）》注册营养师考点串讲｜温故（准则一二三四）→ 新知 → 口诀 → 易错 → 解题 → 提问 → 改编题 → 费曼 → 预告

一、温故（上四课要点快速过）

考点一句话核心关键数字
平衡膳食宝塔五层：谷薯→蔬果→畜禽鱼蛋奶→大豆坚果→油盐盐<5g、油25–30g、奶300–500g
准则一 食物多样谷类为主、每天12种/周25种谷类200–300g（全谷杂豆50–150g、薯类50–100g）
准则二 吃动平衡能量天平、健康体重BMI 18.5–23.9；腰围男≥90女≥85；6000步/周150分
准则三 蔬果奶类全谷大豆餐餐有蔬菜、天天有水果、常吃奶类全谷蔬菜≥300g（深色½）、水果200–350g、奶≥300ml、坚果≈10g
准则四 适量鱼禽蛋瘦肉白肉优先、不弃蛋黄、少吃加工熏腌120–200g/天；鱼≥2次/周；蛋300–350g/周

今日目标：掌握了"吃什么、吃多少"，今天守好"怎么做得清淡、别让盐油糖酒拖后腿"这条底线。

二、导入（一个案例）

案例：大三学生小盐"重口味"——外卖顿顿麻辣香锅、咸菜就白粥、奶茶一天一杯、宵夜炸鸡配啤酒。最近体检：血压 138/88（偏高）、甘油三酯高、有脂肪肝倾向。
老师提问：小盐的吃法，踩了准则五的哪些"坑"？学完今天这一课，你能给他开出几条整改单吗？

三、新知讲解：准则五 少盐少油，控糖限酒

1. 先记"三把尺"——核心数字（考试高频）

图1｜盐<5g、油25–30g、添加糖<50g（最好<25g）、反式脂肪酸<2g、酒精<15g酒精/天。2022版盐从6g收紧到5g、酒精统一为15g。

2. 五条核心推荐（官方原文）+ 考试六条拆分

官方原文 5 条

  1. 培养清淡的饮食习惯，少吃高盐和油炸食品。成年人每天摄入食盐不超过 5g，烹调油 25～30g。
  2. 控制添加糖的摄入量，每天不超过 50g，最好控制在 25g 以下。
  3. 反式脂肪酸每天摄入量不超过 2g。
  4. 不喝或少喝含糖饮料。
  5. 儿童青少年、孕妇、乳母以及慢性病患者不应饮酒。成年人如饮酒，一天饮用的酒精量不超过 15g。

考试常用 6 条拆分

  1. 食盐 < 5g/天，烹调油 25–30g/天；
  2. 添加糖 < 50g/天，最好 < 25g；
  3. 反式脂肪酸 < 2g/天；
  4. 不喝或少喝含糖饮料；
  5. 儿童青少年、孕妇、乳母、慢性病患者不应饮酒；
  6. 成年人如饮酒，一天酒精量 < 15g。

3. 底层逻辑：为什么这样推荐？（关键事实·证据）

  • 高盐：升高血压，增加脑卒中、胃癌、心血管疾病风险；我国居民盐摄入（约10g/天）远超推荐。
  • 高油/高能量：导致超重肥胖、血脂异常、脂肪肝。
  • 添加糖/含糖饮料：增加龋齿、肥胖、2型糖尿病、血脂异常风险（游离糖快速升血糖、促脂肪合成）。
  • 酒精：无安全阈值，损害肝脏，增加多种癌症（口腔、食管、肝、结直肠等）风险。

4. 隐形盐——"不咸≠无盐"（重要易错）

食盐 = 氯化钠（NaCl），钠占约 39%，换算：1 g 钠 ≈ 2.5 g 食盐。很多食物"吃着不咸，钠却很高"。

图2｜酱油、腐乳、挂面、咸菜、薯片、火腿培根都含"隐形盐"。控盐要会看营养标签的"钠"含量，不能只少放盐勺。

5. 隐形糖——"不甜≠无糖""控糖≠戒主食"

图3｜添加糖主要来自含糖饮料、糕点、蜂蜜、果汁；一瓶可乐≈53g糖已超全天上限。控糖控的是"添加糖"，不是主食碳水。

6. 酒精换算——"15g是酒精量，不是酒的量"

公式：酒精量(g) = 酒量(mL) × 酒精度(%) × 0.8（乙醇密度约 0.8 g/mL）。成年人一天酒精量 < 15g，约为：

图4｜酒精量=酒量×酒精度%×0.8；15g≈450mL啤酒/150mL葡萄酒/50mL 38°白酒。注意是"酒精量"不是"酒的量"。

7. 减盐、减油、控糖"三件套"技巧（常考怎么做）

图5｜减盐（量化+替代+出锅放）、减油（定量+蒸煮炖拌）、控糖（不喝甜饮+看标签+水果代果汁）。
图6｜把"高盐高油高糖酒"调成"清淡量化、蒸煮为主、戒甜饮、限酒"。

四、口诀（背下来就是分）

① 三把尺："盐五油三(25–30)糖五零(最好二五)，反式两克酒一五。"

② 隐形盐："不咸也有盐，看钠不看咸；一瓶酱油半瓶盐，挂面薯片偷偷咸。"

③ 酒精："十五克酒精是红线，四类人儿滴酒不沾（儿童青少年·孕妇·乳母·慢性病）。"

④ 三件套："盐勺量化出锅放，葱姜蒜香代盐咸；蒸煮炖拌少油煎，甜饮料儿靠边站。"

五、易错点（6 个，考试最爱挖坑）

易错1：盐的推荐量记成旧版 6g → 2022 版已收紧为 < 5g/天。
易错2：酒精记成旧版"男 25g / 女 15g" → 2022 版统一为 < 15g 酒精/天；且是"酒精量"不是"酒的量"。
易错3：禁酒人群只想到孕妇、乳母 → 还有儿童青少年、慢性病患者，共四类不应饮酒。
易错4：把"添加糖 < 50g"与"最好 < 25g"混为一谈 → 上限 50g，理想目标 25g 以下。
易错5："不咸=无盐""不甜=无糖" → 隐形盐（酱油/挂面/薯片）、隐形糖（可乐/糕点）照样超标。
易错6：把"控糖"误当成"戒主食碳水" → 控的是添加糖/游离糖；烹调油不是越少越好（脂溶性维生素吸收需要适量脂肪）。

六、三步解题法

第一步·读题定位：判断考"量（盐油糖酒数字）/ 隐形盐糖 / 酒精换算与禁酒人群 / 实操技巧"哪一类。

第二步·套数字原则：盐<5g、油25–30g、添加糖<50g（最好<25g）、反式<2g、酒精<15g；四类人禁酒；1g钠≈2.5g盐；酒精量=酒量×酒精度%×0.8。

第三步·辨陷阱：看是否用旧数据（6g/男25女15）、是否把"酒量"当"酒精量"、是否忽略隐形盐糖、是否把"控糖"理解成戒主食。

七、课堂提问（师生互动，先想再对答案）

问1：小盐把每天食盐从 6g 减到 5g，是不是就达标了？
答：还差一步。①是减到 5g 以内（<5g），不是"等于5g就行"；②更要防隐形盐——咸菜、酱油、挂面、薯片里的钠也算进去，总量仍可能超。
问2：小盐宵夜喝了 2 两（100mL）52° 白酒，酒精量多少？超标吗？
答：100 × 52% × 0.8 = 41.6 g，远超 15g 上限近 3 倍，明显超标。
问3："无糖可乐"能不能随便喝？
答：无糖可乐虽不加蔗糖，但准则讲的是"不喝或少喝含糖饮料"；代糖长期大量摄入仍有争议，最稳妥还是白水、淡茶。
问4：高血压患者只在做菜时少放盐勺就够了？
答：不够。还要看隐形盐——酱油、咸菜、挂面、火腿培根的钠，以及学会看营养标签的"钠"含量。
